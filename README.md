# FloodMap

A real-time flood map service that helps users quickly identify flooded areas in Seoul.

## Why?

Current flood information is difficult to understand for ordinary users.

FloodMap focuses on simple, intuitive, real-time visualization.

## Features

- Real-time flood map
- Top-K flooded areas
- Administrative district visualization
- Live updates
- Mobile-friendly UI

## Data Sources

- FloodMap Open API
- Seoul Open Data
- ITS Korea
- Weather API
